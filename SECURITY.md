# Security Policy

## Supported Versions

We support security updates for the latest `main` branch and the most recent release.

## Reporting a Vulnerability

Please report security issues privately.

- Email: security@agentic-dev-library.org
- Include: steps to reproduce, impact, and any relevant logs or PoC.

We will acknowledge reports within 3 business days.

## Disclosure Policy

We coordinate fixes and publish advisories after a patch is available.
